#### Technologies

- React + vite
- Node js
- mongoDB
- socket.io

  
#### Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`ConnectionUrl` = Your MongoDB Connection String
`JWT_SECREAT` = your secure secreat key
